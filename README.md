# What does delebetor do?
- It installs and adjusts the tools we use when performing pentesting.
- It also deletes and cleans the tools we have downloaded.
-----
- [x] ByCh4n Presents!

# Installation:
```bash
git clone https://github.com/ByCh4n/Delebetor.git
cd Delebetor
sudo make install

# Please run "sudo"
```
